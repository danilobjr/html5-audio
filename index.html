HTML5 Audio.

A minimalist jQuery plugin for HTML5 Audio Tag.

  • Global player controls
  • Play song from any element

Usage

  1. Put the CSS on your site

    <link href="html5audio.css" rel="stylesheet" />
<link href="html5audio_animations.css" rel="stylesheet" />

  2. jQuery Library

    <script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.2/jquery.min.js"></script>

  3. HTML5 Audio jQuery Plugin

    <script src="html5audio.js"></script>

  4. Add a container to the global player

    <audio id="player"></audio>

  5. Some audio sources

    <a href="#" class="song"
    data-h5a-song-artist="Artist 1"
    data-h5a-song-name="Music 1"
    data-h5a-song-url="musics/music1.mp3"
    data-h5a-cover-url="images/albumcover1.jpg"
>Music 1</a>
<a href="#" class="song"
    data-h5a-song-artist="Artist 2"
    data-h5a-song-name="Music 2"
    data-h5a-song-url="musics/music2.mp3"
    data-h5a-cover-url="images/albumcover2.jpg"
>Music 2</a>

  6. Configure

    $(function (){
    $('#player').html5Audio({ songSources: '.song' });
});

  7. And... That's it! =]
